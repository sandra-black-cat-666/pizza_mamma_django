🍕 Pizza Mamma

A fun and interactive pizza restaurant website built with Django. Browse the menu, spin the pizza roulette to let fate decide your meal, and discover fun facts about Italian desserts along the way.


✨ Features

🎰 Pizza Roulette: Can't make up your mind? Spin the wheel and let the roulette pick a random pizza for you.
🍮 Italian Dessert Fun Facts Slider: a scrolling carousel of interesting facts about classic Italian desserts.
🍽️ Menu: Browse the full pizza selection with descriptions.



🛠️ Tech Stack

Backend: Python 3.x, Django
Frontend: HTML, CSS, JavaScript
Templating: Django Templates
